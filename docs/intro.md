# 开往-友链接力 2.1.0

## 关于开往

**开往**的生日是 **2020年3月12日**，取自**开**放的**网**络，是一个友链接力项目。

开往的logo是代表“世界”的星环 + 代表“穿梭”的列车 —— 相信你听到“开往”这个词最多的时候一定是在列车上，“由 xx 开往 xx 的列车……”。![example](https://www.travellings.cn/assets/logo-index.gif)

每当有人访问加入开往的网页时，点击“开往“会**随机跳转**到另一个加入开往的网页。之后，再次点击网页上的”开往“或**后退网页**，将继续随机跳转到另一个加入开往的网页。

加入开往的网页越多，友链接力的规模越大，分享流量的规模也越大。

目前已有 1k+ 人关注开往[![GitHub stars](https://img.shields.io/github/stars/travellings-link/travellings?style=social)](https://github.com/travellings-link/travellings/stargazers)，超 900 成员加入过开往，[点此查看](https://list.travellings.cn)成员列表。
