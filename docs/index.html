开往-友链接力 2.1.0

关于开往​

开往的生日是 2020年3月12日，取自开放的网络。

开往的logo是代表“世界”的星环 + 代表“穿梭”的列车 —— 相信你听到“开往”这个词最多的时候一定是在列车上，“由 xx 开往 xx 的列车……”。

每当有人访问加入开往的网页时，点击“开往“会随机跳转到另一个加入开往的网页。之后，再次点击网页上的”开往“或后退网页，将继续随机跳转到另一个加入开往的网页。

加入开往的网页越多，友链接力的规模越大，分享流量的规模也越大。

目前已有900+人关注开往，超800成员加入过开往，点此查看成员列表。

支持开往​

  • 在你的网页上尽可能多的放置开往的徽标，帮助宣传让更多的人加入开往。
